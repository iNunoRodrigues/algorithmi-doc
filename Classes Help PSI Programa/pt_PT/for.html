Ciclo para

A instrução para, ou ciclo para como é mais conhecido, é por norma usado em situações em que sabemos o número de iterações à partida.

Sintaxe geral do ciclo para:

Para (inicializações; condição; pós-instrução)
  Instrução;

O ciclo para funciona da seguinte maneira:

  1º - É executado o código que está nas "inicializações", este código é executado apenas uma única vez.

  2º - A "condição" é avaliada, se o resultado da condição for Falso então o ciclo para termina, caso o resultado da condição seja Verdadeiro então a "instrução" é executada.

  3º - Depois da "instrução" ser executada, é executada a "pós-instrução", por norma é aqui que acontecem as alterações necessárias para passar à próxima iteração.

  4º - Voltar a repetir o 2º passo.
