Operadores Relacionais

Os operadores relacionais têm como objectivo estabelecer relações entre os operandos.

 Operadores disponíveis para números relacionais:

     =  : Igual

     =/=  : Diferente

     >  : Maior

     >=  : Maior ou igual

     <  : Menor

     <=  : Menor ou igual
