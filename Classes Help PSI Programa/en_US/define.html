Definir variáveis

As variáveis são o suporte dos dados em programas computacionais. Representam um espaço na memória do computador que permitem armazenar um determinado tipo de dados que varia ao longo da execução do programa.

Sintax

As variáveis são compostas por:

  • Tipo  : Tipo de dados que são suportados pela variável
    • ​Tipos de dados:
      • ​inteiro - número inteiro - exemplo 1234
      • real - número real - exemplo 3.14
      • texto - conjunto de caracteres - exemplo "Olá Mundo"
      • lógico - valores Verdadeiro ou Falso

    • Tipo Descrição Valores Valor por defeito
    Inteiro Valores ordinais definidos com 4 bytes -2 147 483 648
    2 147 483 647    		 0
    Real Valores com parte decimal definidos com 64 bits -1.7 E 308
    1.7 E 308    		 0.0
    Lógico Valores lógicos de 1 bit Verdadeiro
    Falso    		 Falso
    Texto Conjunto de caracteres "Sequências de caracteres"
    "entre aspas"    		 "" (vazio)

  • Nome : nome pelo qual os dados vão ser acedidos:
    • Regras para a definição de nomes:
      • Só pode conter os caracteres [a..z] , [A..Z], [0..9]
      • Não pode começar por um número [0..9]
      • Não pode ser uma palavra reservada
        • ler, escrever, se, senão, enquanto, etc.
  • Valor : conteúdo da variável. O conteúdo da variável tem de ser compatível com o tipo.

Variáveis compostas

  • Vector : É um conjunto de elementos consecutivos, sendo obrigatoriamente todos do mesmo tipo e é possível aceder individualmente a cada elemento do array.
    • ​Sintaxe geral de um Vector:
      • ​ tipo nome_da_variável [nº de elementos]
        • Exemplo Prático:
          • int v[10] - v é um vector com 10 inteiros.
  • Matriz : É um vector de vectores, ou seja, é um vector com mais de uma dimensão.
    • ​Sintaxe geral de uma Matriz:
      • ​ tipo nome_da_matriz [dim 1] [dim 2] [dim n]
        • Exemplo Prático:
          • char desenho[3] [3]; - desenho é uma matriz com 3 linhas e cada uma com 3 posições.
  • Registos : conteúdo da variável. O conteúdo da variável tem de ser compatível com o tipo.
